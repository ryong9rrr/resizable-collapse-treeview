# Resizable Collapse TreeView

> Electron + React app

## dependencies

- node (v14.16.0)
- npm (v6.14.11)
- react (v18.2.0) + JS
- electron (v23.1.3)

## Getting Started

1\. 의존성 모듈들을 설치해주세요.

- `npm install`

2\. 루트 경로에 `.env` 파일을 생성하고 아래와 같이 작성해주세요.

`BROWSER=none`

3\. `npm run dev` 명령어로 실행해주세요.
